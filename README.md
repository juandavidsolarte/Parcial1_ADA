# Parcial1_ADA
Ejercicio de HackerRank sobre la señal de Marte.

## Cómo ejecutar

1. Abrir una terminal en la carpeta del proyecto.
2. Ejecuta el programa con Python 3:

```bash
python mars_exploration.py
```

3. Escribir una cadena de entrada cuando el programa la solicite y presiona Enter.
Ej:  SOSSPSSQSSOR,SOSSPSSQSSORSOSSOSSPRQ


Luego de ingresar la cadena por teclado y revisar el resultado que corresponde a la cantidad de SOS encontrardos en toda la cadena de texto.
